kiosk¶

api.views.kiosk_zip_code.index(request, zip_code)¶

GET http://<domain>/api/kiosk/<zip_code>

Given a zip code, find other similar zip codes within the same k-means clusters. For these zip codes, rank kiosks based on their distances from Strava segments. Return this ranked list of kiosks as suitable locations for advertisements.

Parameters:
  • request (requests) – HTTP GET Request
  • zip_code (str) – zip code
Returns:

same cluster zip codes and the closest kiosks latitude and longitude to Strava segments
Return type:

dict[list]
